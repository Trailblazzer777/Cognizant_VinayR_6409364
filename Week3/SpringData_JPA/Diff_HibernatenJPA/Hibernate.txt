Hibernate
Type: Framework (ORM Tool)

Implements: JPA Specification.

Key Features:

Handles object-relational mapping (ORM).

Manages caching, fetching strategies, lazy/eager loading.

Supports JPQL and native SQL.

Responsibility: You manage sessions and transactions explicitly unless using Spring.




public Integer addEmployee(Employee employee) {
   Session session = factory.openSession();
   Transaction tx = null;
   Integer employeeID = null;
   try {
      tx = session.beginTransaction();
      employeeID = (Integer) session.save(employee); 
      tx.commit();
   } catch (HibernateException e) {
      if (tx != null) tx.rollback();
      e.printStackTrace(); 
   } finally {
      session.close(); 
   }
   return employeeID;
}
